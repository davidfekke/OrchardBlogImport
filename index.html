Orchard Blog Import.

Use this node program to generate markdown files for your blog. The markdown format is consistent with Jekyll site generators. The query is based on Orchard 1.2 schema format. I am using Tedious to make the SQL Server connection. Just change the IP address, username, password and database to your Orchard db, and let node do the rest.

var config = {
  server: '*server-ip-address*',
  userName: '*username*',
  password: '*password*',
  options: {
    debug: {
      packet: true,
      data: true,
      payload: true,
      token: false,
      log: true
    },
    database: '*Orchard_db_name*'
  }
};
